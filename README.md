Ola Ride Details Dashboard
This project is an Ola Ride Details Dashboard created using Power BI to analyze ride booking data and customer behavior.
The dashboard provides insights into total bookings, ride status, revenue, cancellation details, vehicle types, and customer ratings.
Features
Total ride bookings overview
Successful & cancelled rides analysis
Revenue and booking value tracking
Vehicle type performance
Customer & driver ratings
Monthly and daily booking trends
Interactive filters and charts
Tools Used
Power BI
Excel / CSV Dataset
Power Query
DAX
Project Objective
The main objective of this project is to understand ride booking patterns and visualize important business insights using interactive dashboards.
Files Included
Power BI Dashboard file (.pbix)
Dataset file
Dashboard screenshots
Conclusion
This dashboard helps in analyzing Ola ride operations effectively and supports better business decision-making through data visualization.
